About GLFS

GLFS - GCE Loader For Shaders

GLFS is a tool designed to help you manage and apply shaders to Minecraft Bedrock Edition using BetterRenderDragon.

Features:

  • Easy shader management and application
  • Preset system to save your favorite shader configurations
  • Automatic MaterialBinLoader installation and configuration
  • Modern, user-friendly interface

How to use:

  1. Set up your paths in the Settings tab
  2. Make sure MaterialBinLoader is properly installed
  3. Select a shader from the list and click "Apply Selected"
  4. Launch Minecraft and enjoy your shader!
